# Task Handler

This is a program for a small business to help manage tasks assigned to members of a team.

## Motivation

This project was created to consolidate the knowledge I've gained through various tasks.

## Prerequisites

* Install Python 3.8.
* Open and the run python file with IDLE.

## Functionality

It uses a login system where registered users can see and manage tasks. An admin user is available with more funcionality than regular users. All tasks and users are saved in seperate text files.
